# 🎟️ Face Recognition Ticketing System

얼굴 인식 기반 티켓팅 및 입장 관리 시스템  
불법 암표 거래를 방지하고 공정한 공연/스포츠 관람 문화를 만들기 위한 프로젝트.



## 프로젝트 개요

이 프로젝트는 공연 및 스포츠 경기에서 발생하는 **암표 거래 문제**를 해결하기 위해  
얼굴 인식 기술을 활용한 **본인 인증 기반 티켓팅/입장 시스템**을 구현합니다.

- 티켓 구매자와 실제 입장자의 일치 여부 검증
- 대리 입장 및 티켓 양도 방지
- 공정한 티켓 분배 환경 구축


## 현재 티켓 시장에서는 다음과 같은 문제가 발생합니다:

- 암표 거래로 인해 가격 왜곡 발생
- 비대면 거래로 추적이 어려움
- 대리 입장 및 신분 도용 가능
- 공연/스포츠 산업 신뢰도 저하


## 본 프로젝트는 다음과 같은 방식으로 문제를 해결합니다:

- 얼굴 인식 기반 실시간 본인 인증
- 사용자 정보 및 예매 내역을 데이터베이스로 관리
- 입장 시 얼굴 데이터와 티켓 정보를 매칭
- 대리 입장 및 암표 사용 차단


## 주요 기능

- 🎥 실시간 얼굴 인식
- 🧑 사용자 얼굴 등록 및 인증
- 🎫 티켓 정보 관리
- 🔍 얼굴 데이터와 티켓 매칭 검증
- 🚫 대리 입장 방지 시스템


## 기술 스택

- **Python**
- **OpenCV** – 얼굴 감지 및 이미지 처리
- **face_recognition** – 얼굴 인코딩 및 비교
- **Scikit-learn** – 유사도 계산 (Cosine Similarity)
- **SQLite** – 사용자 및 티켓 데이터 저장
- **Tkinter** – UI 및 알림 처리


## 시스템 구조 (개념)

사용자 얼굴 등록 → DB 저장 → 티켓 구매 → 입장 시 얼굴 인식 → DB 비교 → 인증 여부 판단
